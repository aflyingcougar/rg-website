PUG Mod

One of the latest mods installed on the CS:GO server, Pug Setup v1.4.2, provides users with the ability to start pick-up games with just a few simple chat-commands. The plugin was written by AlliedModders user, splewis. By simply typing .setup into chat, anyone can access an on-screen menu allowing easy changing of match-variables such as team size, map choice, and many others. Author of the plugin, splewis says,

The overall experience is similar to CEVO or ESEA scrim servers, and significant work was done to make it easy to install and use without needing much technical knowledge.
splewis in AlliedModders
How to Use the Mod: Assuming nobody has already started a match, anyone in the server can type .setup to open the menu featured in the picture to the left. From this menu, you can use the numbers on the keyboard to navigate the menus and change any desired matchmaking parameters (team size, map, etc.). When you have altered all the parameters to how you would like, press 1 to finalize the match settings. The server will go into 'matchmaking mode,' awaiting for the required amount of players to type .ready in chat. Once the required amount of players are "ready" the match will automatically (or not automatically) start the match, depending on whether you turned on "Auto-Live" or not.

Upon match end, the server will go back into "warmup" mode, awaiting another .setup command.

Command Description
.setup begins the setup phase and sets the pug leader
.ready marks you as ready
.notready marks you as unready
.pause requests a pause (which takes effect next freezetime)
.unpause request an unpause
.start starts the game if auto-live has been disabled
.capt gives the pug leader a menu to select captains
.rand selects random captains
.leader gives a menu to change the game leader
.endgame force ends the game safely (only the leader can do this, note that this **resets the leader** to nobody)
.stay chooses to stay after winning a knife round
.swap chooses to swap after winning a knife round

TeamSpeak 3 Server

As of now, we currently locally host a 16-slot TeamSpeak 3 server. For those who don't know, TeamSpeak is a VOIP (Voice Over IP) program used to communicate with your friends outside of the games you normally play. The main advantage to using TeamSpeak (TS) over using in-game voice is the higher audio quality and you can communicate at all times (through loading screens). In other words, even if you haven't loaded up Counter-Strike yet, and you're staring at your desktop, you can still be talking to your friends in TS.

Currently the 16-slot TS server is hosted on the owner's, bpEX, computer. As our community grows we plan to upgrade our TeamSpeak 3 server to more slots and to also pay to have it hosted on a more reliable setup. Below we have listed our connection info, and how to connect if it is your first time using TS.

IP: 45.23.180.10
Port: 9987

How to connect:
If you havent downloaded the TeamSpeak 3 client, you can do so in the Links section below. Afer downloading and installing TeamSpeak 3, it will prompt you to go through a "Setup Wizard." This will get you set up with your recording and playback devices, as well as the buttons (or voice-activation) that you will use to talk in TS. Now you can follow the steps below to Bookmark our server and connect:


  • 1. Go to the Servers page on our website
  • 2. Press the 'Join Server' button to the right.
  • 3. In the window that pops up, press 'Bookmark and Connect'
